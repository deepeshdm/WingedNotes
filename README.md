## Setup Locally 👩‍🔧

1. Git clone the project repository on your local system
```javascipt
git clone https://github.com/deepeshdm/WingedNotes.git
```

2. Install dependencies in package.json
```javascipt
cd WingedNotes
npm install
```

3. Deploy project on local server
```javascipt
npm run build 
num run start
```
